# Cybersiglism Images Setup

## Where to Find Cybersiglism Tattoo Images (Free/Transparent)

### Best Sources:

1. **Pinterest** (search: "cybersiglism tattoo", "biomechanical tattoo", "organic tattoo flash")
   - Save images and use remove.bg to make transparent

2. **Freepik** (freepik.com)
   - Search: "tattoo vector", "tribal tattoo"
   - Download as PNG with transparent background

3. **Vecteezy** (vecteezy.com)
   - Search: "tattoo art", "biomechanical"
   - Free vectors with transparent backgrounds

4. **Flaticon** (flaticon.com)
   - Search: "tattoo", "tribal"
   - Download as PNG

5. **Remove.bg**
   - Use to remove backgrounds from any tattoo images you find

## Recommended Image Types:

- Flowing organic lines
- Biomechanical spines
- Tribal patterns
- Vein/nerve-like structures
- Bone-like shapes
- Tendril/tentacle patterns

## Setup Instructions:

1. **Download 6 cybersiglism images** (PNG with transparent background)
2. **Rename them to:**
   - `sigil-1.png`
   - `sigil-2.png`
   - `sigil-3.png`
   - `sigil-4.png`
   - `sigil-5.png`
   - `sigil-6.png`

3. **Place in:** `/public/images/symbols/`

4. **The images will automatically:**
   - Be displayed with purple tint (matches the site)
   - Have glow effects
   - Fade in with animations
   - Position around the hero section

## Image Specs (Recommended):
- Format: PNG with transparency
- Size: 500x500px to 1000x1000px
- Style: Black ink on transparent background
- Content: Organic, flowing, biomechanical patterns

## Current Setup:

The code expects images at these paths:
```
/public/images/symbols/sigil-1.png  (top-left)
/public/images/symbols/sigil-2.png  (top-right)
/public/images/symbols/sigil-3.png  (bottom-left)
/public/images/symbols/sigil-4.png  (bottom-right)
/public/images/symbols/sigil-5.png  (left-middle)
/public/images/symbols/sigil-6.png  (right-middle)
```

If images are missing, the site will still work - they just won't display.

## Pro Tips:

- Images automatically get purple hue-rotate (looks better on dark background)
- Add glow effects behind each image
- Images pulse and fade in for dynamic effect
- Positioned around the page edges so they don't block content
